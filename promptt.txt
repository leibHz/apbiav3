Classes Bootstrap a REMOVER:
Container: container, container-fluid → CSS Grid/Flex personalizado
Grid: row, col-* → CSS Grid
Botões: btn, btn-primary, btn-* → .btn customizado
Cards: card, card-body, card-header → .card customizado
Badges: badge, bg-* → .badge customizado
Tables: table, table-hover → .table customizado
Forms: form-control, form-group, form-label → customizado
Alerts: alert, alert-* → .alert customizado
Modals: modal, modal-* → .modal-overlay customizado
Utilities: d-flex, text-center, mb-3, etc → CSS direto
